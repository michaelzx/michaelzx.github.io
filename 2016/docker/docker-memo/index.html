个人Docker使用备忘录

常用命令（个人收藏）

为Alpine安装bash

docker exec <容器名orID> apk add --no-cache --update-cache bash

进入容器bash

docker exec -it <容器名orID> /bin/bash

大保健

用一行命令大扫除：

docker kill $(docker ps -q) ; docker rm $(docker ps -a -q) ; docker rmi $(docker images -q -a)

查看 docker中对应容器的 IP

docker inspect 容器ID
docker inspect 容器ID | grep IPAddress

容器同步宿主机时间

非常easy的方法，只要给容器加上一个volume：

/etc/localtime:/etc/localtime:ro
